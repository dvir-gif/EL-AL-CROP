סקירה

מה משתנה ב 1 ביולי

מ 1 ביולי משתנה אופן התימחור והכרטוס של הסכמי החברות. ההסכם הוא חוזה גלובלי אחד, מתומחר ומכורטס ב IATA הישראלית בלבד כך שההנחה מוחלת אוטומטית והכרטוס מתבצע עם קוד החברה.

תשלום והערות

אמצעי תשלום ודגשים נוספים

ה FOP נשמר כפי שאתם מכניסים אותו היום. שימו לב להבדל בין כרטוס ב UATP לבין הסכם עם שובר זיכוי.

כרטוס עם UATP UATP

להכניס את ה FOP כפי שעושים היום. יש לוודא שה FT נותר תקין ומכיל את קוד העמלה ואת מספר החברה.

לוודא נוכחות: COM11111 + מספר החברה

הסכם עם שובר זיכוי CCTP

ה FOP נותר זהה למה שמוכנס היום, בפורמט הבא:

AGT*CCTP11459000000%COM11111

דגשים לכרטוס ידני / קבוצות

  • 1
    בתמחור ידני (למניעת ADM) עדכנו בשדה Fare Basis 2 את קוד התמחור של החברה, ובשדות Endorsement ו Tour Code את מספר החברה.
  • 2
    שימוש במחירון Publish כדי שהכרטיס ייזקף לטובת החברה, עדכנו בשדה Tour Code את מספר החברה (לדוגמה: COM12345).
  • 3
    קבוצות יש להוסיף קוד חברה במערכת GROUPIT.
  • 4
    חובה בכל הזמנה עסקית לעדכן קוד דרוג DS/DG/DP בצירוף מספר החברה.
